# Contact Form Validation

A responsive contact form built using **HTML**, **CSS**, and **JavaScript** with client-side form validation.

## Features

- Responsive design
- Name validation
- Email validation using Regular Expressions (Regex)
- Message validation
- Displays error messages for invalid inputs
- Shows a success message after successful validation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
contact-form-validation/
│── index.html
│── style.css
│── script.js
│── README.md
```

## Validation Rules

- Name field cannot be empty.
- Email field cannot be empty.
- Email must be in a valid format using Regex.
- Message field cannot be empty.

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open `index.html` using **Live Server**.
4. Fill in the form and test the validations.

## Output

- Displays validation errors for empty or invalid fields.
- Shows **"Form submitted successfully!"** after valid input.
